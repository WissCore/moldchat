# Support

## Where to ask

| What you need                              | Where to go                                                                         |
|--------------------------------------------|--------------------------------------------------------------------------------------|
| Question about using MoldChat              | [GitHub Discussions](https://github.com/WissCore/moldchat/discussions)              |
| Idea or feature request                    | [GitHub Discussions: Ideas](https://github.com/WissCore/moldchat/discussions/categories/ideas) |
| Bug report (reproducible)                  | [GitHub Issues](https://github.com/WissCore/moldchat/issues)                         |
| Security vulnerability                     | [Private Vulnerability Report](https://github.com/WissCore/moldchat/security/advisories/new) — see [SECURITY.md](SECURITY.md) |
| News and announcements                     | [Twitter / X @MoldChatHQ](https://x.com/MoldChatHQ), [Telegram channel](https://t.me/moldapp) |
| Press, partnerships, legal                 | `alan@wisscore.com`                                                                  |
| General contact form                       | [moldchat.com/about#contact](https://moldchat.com/about#contact)                     |

## What we cannot help with

- Recovering messages or accounts. By design, MoldChat servers cannot read your data. Lost device + lost recovery key = lost account.
- Government data requests beyond what is published in our transparency disclosures.
- Custom builds or feature support for closed-source forks.

## Response expectations

We are a small team. Discussions and bug reports are answered as time permits. Security reports are prioritised and answered within 72 hours.
