# Architecture Decision Records

Format: [MADR 4.0](https://adr.github.io/madr/).

File names: `NNNN-kebab-title.md` (zero-padded, sequential).
Statuses: Proposed, Accepted, Deprecated, Superseded.

Once an ADR is Accepted, it is immutable. Subsequent decisions that change direction supersede it via a new ADR that links back.
